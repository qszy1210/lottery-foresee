# 收敛性与差异分析设计

## 目标

- 对比「预测号码」与「实际开奖」之间的差异。
- 在多期数据上统计命中情况，观察模型指标是否随样本量增加而收敛。

## 核心指标

- 命中个数分布（红球/蓝球、前区/后区）。
- 命中率曲线（随期数增长的累计命中率）。
- 预测组合的形态分布（和值、奇偶比、大小比等）与真实开奖分布的差异。

## 实现思路（后端）

- 基于历史数据，对某一时间段进行回测：
  - 使用「第 N 期之前」的历史数据生成下一期预测；
  - 与真实第 N 期开奖号码对比，记录命中情况与特征差异；
  - 在多个 N 上重复上述过程，形成时间序列。
- 对上述结果进行统计汇总，输出：
  - 每期的命中详情；
  - 区间内的整体命中率与特征分布差异；
  - 可用于前端绘图的结构化数据。

## 前端展示建议

- 折线图：累计命中率 / 滑动窗口命中率。
- 条形图：命中个数分布。
- 对比图：预测与实际在和值、奇偶、大小等维度的分布差异。
